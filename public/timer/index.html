Session Timer

45-minute meditation session timer with phase tracking

How to Use

  1. Choose session length: 30, 45, or 60 minutes
  2. Click Start: Timer begins, showing current phase
  3. Visual cues: Progress bar and phase indicators
  4. Audio bell: Chimes at each phase transition
  5. Pause/Resume: If needed during session

Session Structure (45-min)

  • 0:00-5:00 - Grounding (body, breath, heart)
  • 5:00-20:00 - Heart Activation (compassion practice)
  • 20:00-40:00 - Silent Integration
  • 40:00-45:00 - Closing (gentle return)
