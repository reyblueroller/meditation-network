## What is DMN?

The Distributed Meditation Network is an open registry of peer meditation groups worldwide. It provides infrastructure for coordinating meditation sessions without centralized authority, fees, or dogma.

### Key Features

- **Searchable directory** of meditation nodes by location
- **Standardized practice protocol** based on heart-centered compassion
- **Virtual node support** for remote participation
- **Session coordination tools** (timers, schedules)
- **GitHub-based coordination** - transparent and decentralized

## Status

**Alpha** (January 2026) - Initial infrastructure and protocol development

## How It Works

1. **Nodes register** by submitting a YAML file via pull request
2. **Directory auto-generates** via GitHub Actions
3. **Practitioners find nodes** using the searchable directory
4. **Sessions coordinate** using shared protocols and tools
