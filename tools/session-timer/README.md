# Session Timer

Web-based meditation timer for Heart Protocol sessions.

## Features

- **Preset sessions**: 30, 60, and 90 minute configurations
- **Custom sessions**: Adjust phase durations
- **Phase tracking**: Visual display of current practice phase
- **Progress bar**: See session progress at a glance
- **Bell sounds**: Audio cues for transitions (opening, phase changes, completion)
- **Pause/resume**: Full playback controls
- **Screen sharing friendly**: Large, clear display for virtual sessions

## Usage

### Local Use

Simply open `index.html` in a web browser.

### Virtual Sessions

Share your screen showing the timer page during online meditation sessions.

### Custom Sessions

1. Click "Custom ⚙️"
2. Adjust phase durations
3. Enable/disable sound notifications
4. Click "Apply Custom"
5. Start session

## Heart Protocol Structure

**60-minute session (default):**
- Settling: 5 min
- Heart Focus: 20 min
- Compassion Expansion: 25 min
- Integration: 5 min
- Closing: 5 min

**90-minute session:**
- Settling: 5 min
- Heart Focus: 20 min
- Compassion Expansion: 50 min
- Integration: 5 min
- Closing: 10 min

**30-minute session (short):**
- Settling: 3 min
- Heart Focus: 10 min
- Compassion Expansion: 12 min
- Integration: 3 min
- Closing: 2 min

## Technical Details

- Pure HTML/CSS/JavaScript (no dependencies)
- Uses Web Audio API for bell sounds
- Responsive design
- Works offline
