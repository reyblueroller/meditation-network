## How to Use This Directory

**Joining a node:**
1. Find a node near you below
2. Click "Contact Node"
3. Introduce yourself via GitHub Discussions
4. Attend your first session

**No node nearby?**  
[Start your own node](/docs/getting-started/) - it's easier than you think.
